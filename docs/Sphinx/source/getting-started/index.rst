Getting Started
===============

So you want to learn Embedded Software? This is no easy feat, but here are some
resources to get you started. If you are new to the software world, consider
paying a visit to the :doc:`embedded-primer`. Or, if you just want to set up
your computer and jump right in, you can skip ahead to
the :doc:`installation-guide`.
